# Análisis de datos de tiendas

## Descripción

En este proyecto se realizó un análisis de datos de ventas de cuatro tiendas con el objetivo de identificar cuál de ellas debería vender el Sr. Juan. Para llegar a una conclusión se analizaron diferentes factores relacionados con el rendimiento de cada tienda.

El análisis se desarrolló utilizando Python en Google Colab y se trabajó principalmente con la librería Pandas para el manejo de datos y Matplotlib para la creación de gráficos.

## Análisis realizados

Durante el desarrollo del proyecto se realizaron los siguientes análisis:

- Facturación total de cada tienda.
- Ventas por categoría de productos.
- Calificación promedio de los clientes por tienda.
- Producto más vendido y producto menos vendido.
- Costo de envío promedio por tienda.

Estos análisis permitieron comparar el desempeño de cada tienda y observar patrones en las ventas y en la satisfacción de los clientes.

## Visualización de datos

Para facilitar la interpretación de los resultados se generaron varios gráficos utilizando Matplotlib. Se utilizaron diferentes tipos de gráficos, como:

- Gráfico de barras para comparar los ingresos de las tiendas.
- Gráfico de líneas para observar tendencias en los datos.
- Gráfico de dispersión para analizar relaciones entre variables.

Las visualizaciones ayudan a entender mejor el comportamiento de las ventas y hacen más clara la comparación entre las tiendas.

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Cómo ejecutar el proyecto

1. Abrir el notebook en Google Colab.
2. Ejecutar las celdas de código en orden.
3. Revisar los resultados obtenidos en las tablas y gráficos.
4. Leer el informe final para conocer la conclusión del análisis.

## Conclusión

Después de analizar los datos de las tiendas se pueden observar diferencias en los ingresos, en las categorías de productos más vendidas y en las calificaciones de los clientes. Con base en esta información se puede tomar una decisión informada sobre cuál tienda sería la mejor opción para vender.
